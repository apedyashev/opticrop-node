opticrop-node
=============

node.js port of [jueseph/Opticrop](https://github.com/jueseph/Opticrop)

Installation
-------------
on Debian/Ubuntu
```
sudo apt-get install libgd2-xpm-dev 
npm install opticrop-node
```

on Mac OS/X
```
brew install gd
npm install opticrop-node
```
